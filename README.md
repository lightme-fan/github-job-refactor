## Features

- Search fields.
- Filter jobs by location, title, zip code.
- A list of GitHub jobs with pagination.
- Job details page.

```
    - git clone https://github.com/lightme-fan/github-job-refactor.git

    - cd github-job-refactor

    - yarn install or npm install

    - yarn start
```
